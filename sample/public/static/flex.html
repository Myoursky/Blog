7.flex-grow

flex-grow的作用是分配父容器的剩余空间,如果子元素设置的值相等，则等比分配。否则则按设置值所占整体比例来分配。

示例2: 容器宽度为850px。子元素原始大小为40px。margin总和为50px。则剩余空间 = 850 - 50 - 40 * 5 = 600px。然后平分为6份。第二个子元素占两份。所以得到排版如下。

8.flex-shrink

default value: 1

flex-shrink意为当子元素超过父元素宽度时，按比例压缩子元素。如果设置为0，则该元素不压缩。此处父元素宽度为500px。出去margin总和70px。和shrink0占据的100px。还剩于330px。给剩下的6个子元素平分，每个子元素分配到55px。

9.flex-basis

default value: auto,作用类似于width和height,但优先级高于width和height。

10.flex

flex-grow,flex-shrink和flex-basis的缩写形式。默认值为(0 1 auto),可以设置为auto（1 1 auto）和none（0 0 auto）。
